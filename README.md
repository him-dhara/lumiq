## Brand System

| Role | Token | Hex |
|------|-------|-----|
| Background | BG | `#FDFCFC` |
| Card Surface | Card | `#F5F3F1` |
| Foreground | Text | `#1A1917` |
| Border | Divider | `#EDEBE8` |
| Accent - Flame | Primary 1 | `#ED3F27` |
| Accent - Cobalt | Primary 2 | `#1A3AE8` |
| Accent - Amber | Primary 3 | `#FEB21A` |

**Gradient:** `#ED3F27` to `#FEB21A` - left to right on primary CTAs. Hover reversal.

**Typography:**

| Role | Font |
|------|------|
| Display / Headings | `Instrument Serif` - editorial serif, warmly cinematic, gives the brand its photographic feel |
| Body / UI | `Geist` - clean, precise, readable at every size |
| Metadata / Mono | `Geist Mono` - used for EXIF-style labels and technical copy |

**Icons:** Phosphor Icons - duotone weight throughout.
